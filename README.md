# User Revenue Prediction

This repo provides different experiments for building a model for user revenue prediction.

To set up the environment (in case if you want to follow all the steps) just run this command:

```sh
pip install -r requirements.txt
```

Quick overview of folders and files:
- `main.ipynb` file contains all necessary information about analyzing the input dataframe and building final model;
- `user_revenue_prediction.ipynb` file contains information about how to use final model for user revenue prediction on new data;
- `logs` folder contains all necessary log files;
- `data` folder contains all necessary data files;
- `pictures` folder contains all necessary pictures files;
- `utils` folder contains all necessary script files;
- `models` folder contains all models files.
